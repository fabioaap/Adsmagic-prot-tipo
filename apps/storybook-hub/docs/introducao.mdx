# Design System Adsmagic

Hub unificado de tokens, componentes multi-stack e fluxos operacionais para acelerar a migracao do prototipo legado e garantir consistencia visual.

## Proposito

- Garantir linguagem visual unificada em qualquer front-end.
- Facilitar onboarding de devs, designers e QA.
- Suportar evolucao controlada do design system.

## Quickstart

```powershell
.\start-all-storybooks.ps1
```

- Hub: [http://localhost:6006/](http://localhost:6006/)
- React: [http://localhost:6008/](http://localhost:6008/)
- Vue: [http://localhost:7007/](http://localhost:7007/)

Sem PowerShell?

```bash
npm run dev:react
npm run dev:vue
npm run dev
```

## Responsabilidades dos pacotes

- `@adsmagic/tokens`: tokens em CSS (`css/base.css`) e TypeScript (`src/index.ts`).
- `@adsmagic/react`: componentes React com stories e testes.
- `@adsmagic/vue`: paridade Vue 3 com slots e vitest.

## Como usar o hub

1. Consulte o Guia de Tokens antes de criar novos estilos.
2. Explore bibliotecas React/Vue para entender props e variantes.
3. Siga fluxos de operacao para build, versionamento e publicacao.
4. Execute o checklist de qualidade antes dos PRs.
5. Registre decisoes de arquitetura e discovery na secao de governanca.
