# Design System Adsmagic

Este Storybook centraliza tokens, componentes e fluxos de referência.

- **Tokens**: publicados via pacote `@adsmagic/tokens` e consumidos por React e Vue.
- **Componentes React**: acesse em *React Components*.
- **Componentes Vue**: acesse em *Vue Components*.
- **Fluxos e guidelines**: mantenha aqui checklists, padrões e anotações de UX.

## Como navegar

1. Suba os Storybooks específicos com `npm run dev:react` e `npm run dev:vue`.
2. Abra `npm run dev` para este hub e navegue pelos atalhos nas seções laterais.
3. Adicione novas páginas MDX em `apps/storybook-hub/docs` para documentar padrões.
