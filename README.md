# Protótipos do Adsmágic

Este repositório concentra protótipos de interfaces e fluxos.

## Integrações

- [Configuração do MCP do Figma](docs/figma-mcp.md): tokens, manifestos e fluxo para extrair HTML/CSS de frames via MCP.
